What is Css?

CSS to help how HTML elements are to be displayed on screens.

Css Structure

Css Syntax

        
            body {
                margin:    0;
                padding:   0;
            }
            
            h1 {
                font-family: monospace;
            }
            
            p {
                font-size: 14px;
            }

Css Class Selector

        
            .container {
                margin: 20px;
                padding: 20px;
            }
            
            .vurgulu{
                font-weight: bold;
                background: grey;
                color: red;
            }

Css Id Selector

        
            #container {
                margin: 20px;
                padding: 20px;
            }
            
            #vurgulu{
                font-weight: bold;
                background: grey;
                color: red;
            }
