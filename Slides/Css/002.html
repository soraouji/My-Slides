How to use Style Sheets?

External Style Sheet
Internal Style Sheet
Inline Style

Usage of Css External 

            
<head>
    <link rel="stylesheet" type="text/css" href="style.css">
</head>

Usage of Css External 

            
    Content of Style.css file
    _________________________
    
    body{
        background-color: linen;
    }
    
    h1{
        color: maroon;
        margin-left: 40px;
    }

Usage of Css Internal 

            
    <head>
        <style>
            body{
                background-color: linen;
            }
    
            h1{
                color: maroon;
                margin-left: 40px;
            }
        </style>
    </head>

Usage of Inline Css 

            
    <body>
        <h1 style="color:blue;margin-left:30px;">
            Birinci başlık
        </h1>

        <p style="color:grey;margin:15px;">
            Burası bilginin olduğu bir paragraftır.
        </p>
    </body>
